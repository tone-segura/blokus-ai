## Blokus-0405.py
###内容
较为完整的实现了程序的主要部分.

包含了棋盘类和棋子类，在其上定义了一些基本的操作，核心部分的评分函数已省略.

只需简单修改评分函数，就能实现自己的 `BlokusHacker`.
